# SelfForge v0 工作区

本目录按 major 版本聚合工作区内容。小版本更新不再创建新的工作区目录，只在 forge 聚合文件中追加记录。

# 顶层结构

- `source/`：存放受控生成或待验证的源码。
- `tests/`：存放工作区内的测试、样例和夹具。
- `sandbox/`：存放运行时临时执行目录，运行记录按 run id 分层。
- `artifacts/`：存放可保留的构建产物和验证产物。
- `logs/`：存放本地原始日志，摘要必须写入 forge。

# 约束

- 根目录只允许固定入口文件和固定一级目录。
- 禁止将生成文件直接堆放在工作区根目录。
- 认知类记录必须写入 forge，不允许散落在 workspace。
