# 目标

# 计划（Plan）

1. 读取 v1 记忆、任务、版本记录和状态。
2. 明确本轮边界：只生成候选版本，不实现完整进程启动、切换和回滚。
3. 新增结构化状态读写能力。
4. 新增候选版本生成引擎。
5. 将 Supervisor 和 CLI 接入候选生成入口。
6. 编写单元测试、边界测试和错误测试。
7. 执行格式化、编译检查、测试、运行时验证和真实候选生成。
8. 记录错误、版本变化和可复用经验。
