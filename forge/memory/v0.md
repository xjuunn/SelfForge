# 优化建议

下一步应在 `agent-step` 执行前按当前步骤 Agent 自动领取任务板任务，并在工具执行成功、失败或阻断时分别完成、释放或保留任务租约，将步骤状态和任务状态绑定起来。

# 可复用经验

长期复用的 major 级协作文件需要在版本提升后刷新版本线索，但不能覆盖任务、事件和租约状态。会话上下文应保存任务板路径，避免后续执行阶段重新推断队列位置。
