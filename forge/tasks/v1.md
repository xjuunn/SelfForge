# 任务来源

用户指定第一个任务：完成 SelfForge 系统的最基础架构。

# 任务描述

建立一个最小可运行、可验证、可审计的 SelfForge v1 基础骨架。该骨架必须包含标准目录结构、持久化状态、forge 文档归档入口、Rust CLI、Runtime 验证层、Supervisor 编排层，并通过测试确认初始化与验证行为稳定。

# 输入

- 顶层 SelfForge 系统提示词与项目约束
- 当前空白 Rust crate

# 输出

- 标准目录结构
- v1 workspace
- state/state.json
- forge/tasks、forge/memory、forge/errors、forge/versions 文档
- Rust CLI: init、validate、status
- Rust 单元、边界、错误测试

# 计划（Plan）

1. 读取现有仓库与历史 forge 记忆。
2. 建立 v1 任务文档。
3. 实现最小 Rust 架构：layout、runtime、supervisor、CLI。
4. 生成基础目录与持久化状态。
5. 编写单元、边界、错误测试。
6. 执行 Rust Runtime 初始化与验证。
7. 执行测试并记录结果。
8. 写入 v1 memory 与 versions 文档。
9. 若测试通过，按规范提交代码。
