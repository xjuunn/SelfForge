# 任务来源

用户指定第一个任务：完成 SelfForge 系统的最基础架构。

# 任务描述

建立最小可运行、可验证、可审计的 SelfForge v1 基础骨架。

# 输入

- 顶层 SelfForge 系统约束。
- 当前 Rust crate。

# 输出

- 标准目录结构。
- v1 工作区。
- `state/state.json`。
- `forge/tasks`、`forge/memory`、`forge/errors`、`forge/versions` 文档。
- Rust CLI：`init`、`validate`、`status`。
- Rust 单元测试、边界测试、错误测试。

# 计划（Plan）

1. 读取仓库结构。
2. 建立 v1 任务文档。
3. 实现最小 Rust 架构。
4. 生成基础目录和持久化状态。
5. 编写测试。
6. 执行初始化与验证。
7. 写入记忆和版本记录。
8. 测试通过后提交。
