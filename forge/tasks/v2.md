# 任务来源

用户要求继续完成 SelfForge，并优先完成自我进化功能。

# 任务描述

实现 SelfForge 的最小自我进化切片：由 Supervisor 触发生成下一候选版本。候选版本必须拥有独立工作区、forge 归档入口和持久化状态记录。

# 输入

- 当前稳定版本：v1
- 当前状态文件：state/state.json
- 历史记忆：forge/memory/v1.md
- 用户目标：优先推进自我进化能力

# 输出

- Rust 自我进化模块。
- CLI 命令：`evolve`。
- 候选版本目录：`workspaces/v2`。
- 候选版本文档：`forge/memory/v2.md`、`forge/tasks/v2.md`、`forge/errors/v2`、`forge/versions/v2.md`。
- 更新后的 `state/state.json`。

# 计划（Plan）

1. 读取 v1 记忆、状态和现有 Rust 架构。
2. 新增候选版本生成模块。
3. 由 Supervisor 暴露候选版本生成入口。
4. 写入候选版本工作区与 forge 文档。
5. 更新状态为候选已准备。
6. 增加测试覆盖正常生成、版本号边界和非法状态错误。
7. 执行测试和验证。
8. 记录结果并提交。
