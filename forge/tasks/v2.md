# 任务来源

用户要求优先推进 SelfForge 自我进化能力。

# 任务描述

从当前稳定版本 v1 生成下一候选版本 v2，但不切换当前稳定版本。

# 输入

- 当前稳定版本：v1
- 用户目标：优先推进自我进化候选版本生成
- 状态文件：state/state.json

# 输出

- workspaces/v2
- forge/memory/v2.md
- forge/errors/v2
- forge/versions/v2.md
- 更新后的 state/state.json

# 计划（Plan）

1. 读取状态。
2. 验证当前稳定版本。
3. 计算下一版本号。
4. 生成候选版本目录和文档。
5. 验证候选版本。
6. 持久化候选状态。
