# v2

# 版本变化

- 从 v1 生成 v2 候选版本。
- 新增 workspaces/v2 作为候选版本工作区。
- 新增 forge/memory/v2.md、forge/tasks/v2.md、forge/errors/v2、forge/versions/v2.md。
- 更新 state/state.json，记录 candidate_version=v2 与 candidate_workspace=workspaces/v2，current_version 仍保持 v1。

# 新增功能

- EvolutionEngine：读取状态、验证当前版本、计算下一版本号、生成候选版本、验证候选布局、更新持久化状态。
- ForgeState：使用 serde_json 结构化读写 state/state.json。
- Supervisor：新增 prepare_next_version 与 verify_version。
- Runtime：新增按版本验证布局的能力。
- CLI：新增 evolve [goal] 命令。
