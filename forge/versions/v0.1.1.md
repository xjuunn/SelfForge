# v0.1.1

# 版本变化

- 引入 `vMAJOR.MINOR.PATCH` 版本规则。
- 默认进化策略由代际递增改为 patch 递增：`v0.1.0 -> v0.1.1`。
- state/state.json 增加 `version_scheme` 与 `legacy_versions`。
- CLI `evolve` 支持 `--patch`、`--minor`、`--major`，默认 `--patch`。
- 生成 `v0.1.1` 候选版本，当前稳定版本仍为 `v0.1.0`。

# 新增功能

- 语义化版本解析、校验和递增。
- evolve 输出 commit version。
- 候选文档存在时保留，避免覆盖先写入的任务计划。

# 修复内容

- 修复格式化差异。
- 修复候选生成覆盖已有任务文档的问题。
