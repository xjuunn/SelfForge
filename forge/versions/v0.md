# v0 版本记录

# 记录规则

- 本文件集中记录 v0 大版本下的小版本变化。
- 小版本更新只追加到本文件，禁止创建新的小版本版本文件。

## v0.1.7

# 版本变化

- 从 v0.1.6 生成 v0.1.7 候选版本。
- 复用 major 工作区 `workspaces/v0`，不再创建小版本工作区目录。
- 小版本记忆、任务、错误、版本记录均追加到 major 聚合文件。
- `state/state.json` 已将 `current_version` 提升为 `v0.1.7`，并将当前工作区规范化为 `workspaces/v0`。

# 新增功能

- major 聚合工作区规则。
- major 聚合 forge 文档规则。
- 旧状态工作区规范化规则。
- 防止小版本独立目录和独立归档文件的测试覆盖。

# 修复内容

- 减少小版本目录和文件爆炸。
- 修复候选归档追加时的路径类型错误。
- 修复格式检查发现的 rustfmt 差异。

## v0.1.8

# 版本变化

- 从 v0.1.7 生成并提升 v0.1.8。
- 继续复用 major 工作区 `workspaces/v0`，未创建小版本工作区目录。
- 为 `workspaces/v0` 增加固定内部骨架，避免生成文件堆积在根目录。
- `state/state.json` 已将 `current_version` 提升为 `v0.1.8`。

# 新增功能

- workspace 固定分区：`source/`、`tests/`、`sandbox/`、`artifacts/`、`logs/`。
- workspace 根目录白名单校验。
- workspace 分区中文 README 与 `.gitignore`。
- workspace 结构错误测试。

# 修复内容

- 降低 major workspace 长期运行后的根目录文件膨胀风险。
